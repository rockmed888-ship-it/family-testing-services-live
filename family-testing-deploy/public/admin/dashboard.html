Drug Testing Records

Non-DOT collection, result entry, verification, and printing

These website forms are for non-DOT testing. DOT-regulated testing must use the current federal Custody and Control Form or Alcohol Testing Form and the required federal process.
Start a New Drug Test Record Enter donor information, choose the specimen, document consent and collection, enter results, save, then print the saved record. Start a Hair Drug Test Opens the same legally cautious workflow with Hair selected and hair-specific collection fields visible. Start a Nail Drug Test Records fingernail or toenail specimen details without claiming a federal nail collection protocol. Saved Records Search, edit, review result status, and print the exact version stored in the system.
